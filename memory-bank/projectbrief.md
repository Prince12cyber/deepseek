# Project Brief: Deep AI

## Project Overview
Deep AI is a free, open-source application built with Next.js and the AI SDK for powerful AI-powered conversations. It's developed by Vercel and serves as a production-ready template for AI-powered chat interfaces.

## Core Requirements

### 1. AI-Powered Chat Interface
- Support for multiple AI model providers (OpenAI, Anthropic, Google, xAI)
- Unified API through Vercel AI Gateway
- Real-time streaming responses
- Multi-modal support (text, code, images, sheets)

### 2. User Experience
- Modern, responsive chat interface
- Dark/light theme support
- Sidebar navigation for chat history
- Artifact system for rich content display
- Real-time message streaming

### 3. Data Persistence
- Chat history storage with Neon Serverless Postgres
- User data management
- File storage with Vercel Blob

### 4. Authentication
- Secure user authentication with Auth.js
- Support for multiple auth providers

### 5. Developer Experience
- TypeScript for type safety
- Modern React patterns with Server Components
- Component-based architecture with shadcn/ui
- Comprehensive testing with Playwright

## Project Goals
- Provide a production-ready chatbot template
- Enable rapid development of AI-powered applications
- Support multiple AI providers with unified interface
- Maintain high code quality and accessibility standards
- Offer seamless deployment to Vercel

## Target Users
- Developers building AI chatbot applications
- Teams looking for production-ready chat templates
- Developers wanting to integrate multiple AI providers
- Users seeking modern, accessible chat interfaces

## Success Criteria
- Complete chat functionality with streaming responses
- Support for multiple AI model providers
- Persistent chat history
- User authentication
- Responsive, accessible UI
- Easy deployment and configuration
